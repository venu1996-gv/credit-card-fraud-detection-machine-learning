# Credit Card Fraud Detection using Machine Learning

This project detects fraudulent credit card transactions using machine learning classification techniques.

## Project Overview

The goal of this project is to build a machine learning model that can identify whether a credit card transaction is genuine or fraudulent. Since fraud detection datasets are usually imbalanced, this project focuses on handling class imbalance and training a classification model to detect fraud accurately.

## Technologies Used

* Python
* NumPy
* Pandas
* Scikit-learn
* Logistic Regression
* Jupyter Notebook

## Project Steps

1. Imported the required Python libraries
2. Loaded the credit card transaction dataset
3. Explored the dataset and checked class distribution
4. Separated legitimate and fraudulent transactions
5. Handled imbalanced data using sampling
6. Split the data into training and testing sets
7. Trained a machine learning classification model
8. Evaluated model performance using accuracy score

## Output

The model predicts whether a transaction is:

* Legitimate Transaction
* Fraudulent Transaction
